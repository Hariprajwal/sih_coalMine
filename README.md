# ⚒️ SIH Coal Mine Safety System

**SIH_CoalMine** is a comprehensive safety and monitoring solution developed for the **Smart India Hackathon**. It addresses the critical challenges of underground mining by integrating real-time environmental sensing with a centralized dashboard for mine supervisors.

## 🚨 The Challenge
Coal mining remains one of the most hazardous professions due to unpredictable gas leaks, structural instability, and lack of real-time communication. This project aims to minimize risk and save lives through technology.

## ✨ Key Features
* **Real-time Hazard Tracking:** Monitors methane levels, CO2 concentration, and temperature.
* **Worker Health Dashboard:** Displays vital stats and location data for underground personnel.
* **Instant Alert System:** Triggers emergency protocols and notifications when safety thresholds are breached.
* **Low-Latency Communication:** Designed to work in bandwidth-constrained mining environments.

## 🛠 Tech Stack
* **Frontend:** React.js / Vite
* **Backend:** Node.js / Python
* **Sensors/Hardware:** Integration for MQ-series gas sensors and DHT sensors.
* **Data Flow:** Socket.io for real-time communication.

## 🏆 SIH Context
This project was developed as part of the Smart India Hackathon to solve the problem statement regarding **Mining Safety and Real-time Monitoring.**
